
# Effect of COVID-19 Lockdown on Air Pollutant Levels in California (2019–2021)

The goal of this project is to explore whether California's 2020 COVID-19 lockdown led to measurable reductions in air pollution, and whether those reductions reversed in 2021. We'll be looking at five pollutants: CO, NO2, O3, PM10, and PM2.5 across California counties.


### **Data**
The data used in this analysis is as follows:
1. [Outdoor Ari Quality Data - U.S. Environmental Protection Agency](https://www.epa.gov/outdoor-air-quality-data/download-daily-data)
2. [California state COVID-19 cases/deaths/tests - California Open Data](https://data.ca.gov/dataset/covid-19-time-series-metrics-by-county-and-state1)

Full data dictionary can be found at 'data' folder in this reposiroty.
